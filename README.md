# Asistente LLM para estudiantes universitarios con TEA

## El trabajo tiene como objetivo el desarrollo de una herramienta conversacional, basada en chat y/o voz, que permita al alumnado universitario con Necesidades Específicas de Apoyo Educativo (NEAE) pueda practicar situaciones sociales habituales dentro del contexto universitario. La propuesta parte de la necesidad de ofrecer un espacio seguro de entrenamiento donde el estudiante pueda ensayar interacciones que, en la vida real, pueden generarle ansiedad, inseguridad o dificultades comunicativas, como por ejemplo hablar con profesorado, integrarse en grupos de trabajo o gestionar conflictos interpersonales.
